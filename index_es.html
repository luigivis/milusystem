Es un módulo que incluye el paquete Control de Horas, una herramienta que automatiza procesos diarios de registro de entrada y salida de colaboradores, arrojando datos que agilizan y optimizan las labores de las áreas de Recursos Humanos y Contabilidad. Incluye un módulo para facturar productos o servicios; en caso de elegir el primero, el sistema proporciona reportes que permiten evaluar el flujo de movimiento de cada artículo, así como un espacio/ventana para visualizar las existencias de estos; sin embargo, si se elige la venta de servicios, los reportes estiman la productividad en rangos de fecha de cada colaborador para/con el servicio.
